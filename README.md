# Crypto Dashboard

Real-time cryptocurrency price tracker with technical indicators and market analysis tools.

## Features
- Live BTC/ETH/SOL price tracking
- Technical indicators (RSI, MACD, Moving Averages)
- Stripe & Coinbase Commerce payments
- Google Analytics integration
- Rewardful affiliate tracking

## Tech Stack
- Next.js 14.2.28
- TypeScript
- Tailwind CSS
- Prisma + PostgreSQL
- Stripe API
- Coinbase Commerce
- CoinGecko API

## Deployment
